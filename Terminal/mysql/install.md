first install homebrew

### install
* check version
    ```
    brew info mysql
    ```
    it shows:
    ```
    mysql: stable X.X.XX
    ```
* choose version
    last version
    ```
    brew install mysql
    ```
    other version
    ```
    brew install mysql@X.X
    ```

### uninstall
* stop mysql
    ```
    brew services stop mysql
    ```
* uninstall
    ```
    brew uninstall mysql
    ```
